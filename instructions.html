Instructions for MINECRAFT

Description:

MINECRAFT is a game where you must try to build whatever you want! You have all kinds of blocks to help you! Build anything and be sure to take screenshots to send to your friends! ENJOY!!! P.S. This was created by Arnav Bansal.



Controls:

Arrow Keys - up, down, left, and right
SHIFT + B - make the size of the block bigger
SHIFT + S - make the size of the block smaller
W - building a brown brick wall block
Y - building a yellow brick wall block
G - building a ground block
L - building a light green grass block
D - building a dark green grass block
T - building a wood trunk block
R - building a roof block
U - building an unique glowing block
C - building a cloud block
Backspace - destroy a specific block
CTRL + SHIFT + R on Windows or COMMAND + R on Mac - deleting every block & restarting build
